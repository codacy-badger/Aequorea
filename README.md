### FAQ	

1. Q：为什么文章搜索`htc`没有结果？

   A：因为一财的 API 搜索对大小写敏感。

2. Q：为什么图片很小加载却不太快？

   A：没办法，官方就没对图片做处理，给的都是大图。

3. Q：为什么这么耗流量？

   A：因为接口返回的都是大图，推荐在流量充足的时候使用。
